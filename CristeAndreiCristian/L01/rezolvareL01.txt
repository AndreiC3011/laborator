Microsoft Office sau MS Office este o suită de programe de birou create de firma Microsoft Corporation, care sunt însoțite și de o aplicație specială suprapusă lor pentru accesarea unitară simplă.
Microsoft Word este un procesor de text inclus în Microsoft Office și unele ediții ale Microsoft Works întrerupt acum. Prima versiune a Word, lansată în toamna anului 1983, a fost pentru sistemul de operare MS-DOS și a introdus mouse-ul computerului pentru mai mulți utilizatori. Word 1.0 ar putea fi achiziționat cu un mouse inclus, deși nu a fost necesar. Ca urmare precedentele LisaWrite și MacWrite , Word pentru Macintoshîncercat să adăugați mai aproape WYSIWYG caracteristici în pachetul său. Word pentru Mac a fost lansat în 1985. Word pentru Mac a fost prima versiune grafică a Microsoft Word. Inițial, a implementat .doc proprietarformatul ca format principal. Cu toate acestea, Word 2007 a depreciat acest format în favoarea Office Open XML , care a fost ulterior standardizat de Ecma International ca format deschis . Suportul pentru formatul de documente portabile (PDF) și OpenDocument (ODF) a fost introdus pentru prima dată în Word pentru Windows cu Service Pack 2 pentru Word 2007.
Microsoft Excel este uneditor de foi de calcul care a concurat inițial cu Lotus 1-2-3 dominantși l-a vândut în cele din urmă. Microsoft a lansat prima versiune de Excel pentru Mac OS în 1985 și prima versiune pentru Windows (numerotată 2.05 pentru a se alinia la Mac) în noiembrie 1987.
Microsoft PowerPoint este un program de prezentare utilizat pentru a crea prezentări de diapozitive compuse din text, grafică și alte obiecte, care pot fi afișate pe ecran și afișate de prezentator sau tipărite pe folii transparente sau diapozitive .
Microsoft OneNote este un program de notare care adună note scrise de mână sau dactilografiate, desene, decupaje de ecran și comentarii audio. Note pot fi partajate cu alți utilizatori OneNote prin Internet sau o rețea. OneNote a fost inițial introdus ca o aplicație independentă care nu a fost inclusă în nicioediție Microsoft Office 2003 . Cu toate acestea, OneNote a devenit în cele din urmă o componentă de bază a Microsoft Office; odată cu lansarea Microsoft Office 2013 , OneNote a fost inclus în toate ofertele Microsoft Office. OneNote este, de asemenea, disponibil ca aplicație web pe Office pe web, oaplicație desktop Windows freemium (și ulterior freeware ), o aplicație mobilăpentru Windows Phone , iOS , Android și Symbian și o aplicație în stil Metro pentru Windows 8 sau o versiune ulterioară.
Microsoft Outlook (nu trebuie confundat cu Outlook Express , Outlook.com sau Outlook pe web ) este un manager de informații personale care înlocuiește Windows Messaging , Microsoft Mail și Schedule + începând cu Office 97; include un client de e-mail, calendar, manager de sarcini și agendă. Pe Mac OS, Microsoft a oferit mai multe versiuni de Outlook la sfârșitul anilor 1990, dar numai pentru utilizare cu Microsoft Exchange Server . În Office 2001, a introdus o aplicație alternativă cu un set de caracteristici ușor diferit numit Microsoft Entourage . A reintrodus Outlook în Office 2011, înlocuind Entourage.
Microsoft OneDrive este un serviciu de găzduire a fișierelor care permite utilizatorilor să sincronizeze fișiere și să le acceseze ulterior de pe un browser web sau un dispozitiv mobil.
Microsoft Teams este o platformă care combină chat-ul la locul de muncă, întâlniri, note și atașamente.
Microsoft Publisher este oaplicație de editare desktop pentru Windows, utilizată în principal pentru proiectarea de broșuri, etichete, calendare, felicitări, cărți de vizită, buletine informative, site-uri web și cărți poștale.
Microsoft Access este un sistem de gestionare a bazelor de date pentru Windows care combină relaționalul Microsoft Jet Database Engine cu o interfață grafică de utilizator și instrumente de dezvoltare software. Microsoft Access stochează date în propriul format bazat pe motorul de bază de date Access Jet. De asemenea, poate importa sau conecta direct la date stocate în alte aplicații și baze de date.
Microsoft Project este oaplicație de gestionare a proiectelor pentru Windows pentru a ține evidența evenimentelor și pentru a crea diagrame de rețea și diagrame Gantt , care nu sunt incluse în nicio suită Office.
Microsoft Visio este oaplicație pentru diagramă și diagramă de flux pentru Windows, care nu este inclusă în nicio suită Office.
Office Lens este un scaner de imagine optimizat pentru dispozitive mobile . Captează documentul (de ex. Carte de vizită, hârtie, tablă albă) prin intermediul camerei și apoi îndreaptă porțiunea de document a imaginii. Rezultatul poate fi exportat în Word, OneNote, PowerPoint sau Outlook sau salvat în OneDrive, trimis prin Mail sau plasat în Photo Library .
Office Mobile este o aplicație mobilă Office unificată pentru Android și iOS, care combină Word, Excel și PowerPoint într-o singură aplicație și introduce noi capabilități precum realizarea de note rapide, semnarea PDF-urilor, scanarea codurilor QR și transferul de fișiere.
Office Remote este o aplicație care transformă dispozitivul mobil într-o telecomandă pentru versiunile desktop ale Word, Excel și PowerPoint.
Microsoft SharePoint este o platformă de colaborare bazată pe webcare se integrează cu Microsoft Office. Lansat în 2001, SharePoint este vândut în principal ca sistem de gestionare și stocare a documentelor , dar produsul este extrem de configurabil și utilizarea variază substanțial între organizații. Versiunile de MS Office până la MS Office 2003 inclusiv folosesc nativ formate de fișier binare, proprietare, cu extensiile ".doc" (pentru Microsoft Word), ".xls" (Microsoft Excel), ".ppt" (Microsoft PowerPoint), etc. Începând cu MS Office 2007 s-a trecut la un format bazat pe XML, incompatibil înapoi (cu versiunile mai vechi), numit Office OpenXML (OOXML).

Microsoft a înaintat la organizația ISO o propunere de standardizare a formatului Office OpenXML (OOXML) - propunere care încă nu a fost adoptată, deoarece a primit critici serioase d.p.d.v. tehnic.

Principalul competitor de piață al formatelor de fișier Microsoft este formatul "OpenDocument Format" (ODF), care a fost adoptat ca standard oficial ISO în anul 2006 (standardul ISO/IEC 26300:2006). Formatul OpenDocument este folosit de o categorie largă de programe, cum ar fi suita liberă OpenOffice.org, Sun StarOffice, IBM Lotus Notes sau KOffice.

Totuși, utilizatorii de MS Office pot utiliza și fișierele de format OpenDocument, cu condiția să instaleze plugin-ul gratuit ODF, creat de firma Sun Microsystems.
